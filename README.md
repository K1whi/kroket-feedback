# Kroket — Retours & bugs

Merci d'utiliser **Kroket**, l'app qui analyse la qualité nutritionnelle des aliments pour chats et chiens 🐱🐶

Ce dépôt sert **uniquement à recueillir vos retours** : signalements de bugs et suggestions de fonctionnalités. Le code de l'application n'est pas hébergé ici.

## 🐛 Signaler un bug

👉 [Ouvrir un signalement de bug](../../issues/new?template=bug_report.yml)

Pour les problèmes d'affichage (alignement, texte, menus), **une capture d'écran** aide énormément.

## 💡 Proposer une idée

👉 [Proposer une fonctionnalité](../../issues/new?template=feature_request.yml)

## 📋 Voir ce qui est déjà remonté

Avant de créer un ticket, jetez un œil aux [issues existantes](../../issues) : vous pouvez ajouter un 👍 ou un commentaire plutôt que d'ouvrir un doublon.

## 📱 Où trouver la version de l'app ?

Onglet **Infos** de l'application → le numéro de version est affiché en bas.

## ✉️ Vous préférez le mail ?

Vous pouvez aussi écrire à **bonjour@kroketapp.com**. GitHub reste néanmoins le moyen le plus efficace pour suivre l'avancement de votre demande.
